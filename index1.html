Kubernetes Command Explorer

kubectl get

Purpose: List resources (pods, deployments, services, etc.).

Example: kubectl get pods

Flags: -o wide, -n namespace

kubectl describe

Purpose: Show detailed information about a resource.

Example: kubectl describe pod pod-name

Flags: -n namespace

kubectl apply

Purpose: Apply configuration from a file.

Example: kubectl apply -f config.yaml

Flags: --dry-run, -n namespace

kubectl delete

Purpose: Delete resources.

Example: kubectl delete pod pod-name

Flags: -n namespace, --force

kubectl logs

Purpose: View logs of a container in a pod.

Example: kubectl logs pod-name

Flags: -f (follow), --tail

kubectl exec

Purpose: Execute a command in a container.

Example: kubectl exec -it pod-name -- bash

Flags: -n namespace
